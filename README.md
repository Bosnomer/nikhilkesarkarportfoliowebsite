## About this site

A single-page, fully responsive portfolio built with **vanilla HTML, CSS and JavaScript** — no build step, no framework. Just open `index.html` or push to GitHub Pages.

### Features
- 🌗 Dark / light theme toggle (remembers your choice)
- ✨ Animated particle/node background that reacts to the cursor
- 📜 Scroll-reveal animations, scroll progress bar and active-section nav highlighting
- ⌨️ Typing effect in the hero and animated stat counters
- 📱 Fully responsive with a mobile menu
- ♿ Respects `prefers-reduced-motion`

### Structure
```
index.html        # all content / markup
css/style.css     # theme tokens, layout, animations
js/main.js        # theme, nav, reveal, counters, typing, particle canvas
img/              # profile photo
```

### Run locally
Open `index.html` directly, or serve the folder:
```bash
python -m http.server 8000   # then visit http://localhost:8000
```
